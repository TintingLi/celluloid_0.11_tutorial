
This is version 0.11.4/hg38 of celluloid. Check your version of celluloid with

 library(help=celluloid)

Check for a newer version at

 https://github.com/mathieu-lemire/celluloid_0.11/blob/master/INSTALL

Install dependencies from Bioconductor first (from within R):

 source("http://bioconductor.org/biocLite.R")
 biocLite("HMMcopy")
 biocLite("copynumber")

Install copynumber38, a modified version of copynumber that allows alignment to hg38:

 install.packages("devtools")
 library(devtools)
 install_github("mathieu-lemire/copynumber38", ref="1.16.0" )

then install celluloid 0.11.4/hg38 from github:

 install_github("mathieu-lemire/celluloid_0.11", ref="hg38" )

Alternatively you can download the source code located at

 https://github.com/mathieu-lemire/celluloid_0.11/releases

and install it with R CMD INSTALL.
